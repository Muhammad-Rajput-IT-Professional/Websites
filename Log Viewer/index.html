Log Viewer
Created By: Muhammad Rajput (MRAJPUT)
Manual mode shows the full file. Click a line, then highlight the exact timestamp text inside the preview and press "Use selection as timestamp sample".
Load one or more log files to begin.
Tip: If the file uses a custom timestamp style, paste one example in the sample field and the viewer will try to recognize the format automatically.
Supported common patterns include syslog month-day-time, ISO 8601 with Z, and timestamps with timezone offsets like 2026-04-27 02:34:24.694 +00:00.
